# Loop Infinito

O [Loop Infinito][l8p] é um blog focado em tecnologias de desenvolvimento web
front-end, como HTML, JavaScript e CSS, em produtividade e lifestyle.
Fundado e mantido por [Almir Filho][almir] e [Caio Gondim][caio].
